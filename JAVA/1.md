## JAVA
* OOP 언어(Object Oriented Programming, 객체 지향)
* class: 내부의 변수(**variable**, attribute, property), 메소드(method)를 통해 프로그램에서 사용 가능한 객체(instance)를 만든다

## 실행 환경
* VS code는 코드 편집기이므로 실행 환경이 따로 필요
* JDK 인스톨: Java Develop Kit > JVM: Java Virtual Machine > JRE: Java Runtime Environment
* Java로 만들어진 소프트웨어는 OS의 dependence에서 동작하지 않음 > VM에서 동작
* Eclipse: JDK 제공
* 현재는 11을 가장 많이 사용, Spring-boot와 연계 시 17 또는 21
+ 17-21 사용 시 11과 호환 X
